# 🚀 Hritvik Saigaonkar — Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

🔗 **Live Site:** [hritviksaiportfolio.netlify.app](https://hritviksaiportfolio.netlify.app)

---

## ✨ Features

- **Dark mode design** with cyan & purple gradient accents
- **Smooth scroll animations** with Intersection Observer
- **Typewriter effect** in the hero section
- **Responsive layout** — works on mobile, tablet & desktop
- **Functional contact form** powered by Formspree
- **Downloadable resume** (PDF)
- **Project showcase** with tech stack tags
- **Sticky navbar** with active section highlighting

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure & Semantics |
| CSS3 | Styling & Animations |
| JavaScript (Vanilla) | Interactivity & Logic |
| Tailwind CSS (CDN) | Utility Classes |
| Formspree | Contact Form Backend |
| Google Fonts | Typography (Inter, JetBrains Mono) |
| Netlify | Hosting & Deployment |
